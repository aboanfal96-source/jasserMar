# 🆕 تحديثات النسخة الجديدة

## 🔧 إصلاحات

### ❌ → ✅ خطأ "txt.replace is not a function"
**السبب:** الكود كان يفترض أن `d.error` نص، لكن Claude API يرجعه ككائن `{type, message}`.
**الحل:** 
- أعدت بناء `api/ai.js` لترجع شكل موحّد `{ok: true, text}` أو `{ok: false, errorMsg}`
- ترجمة كل أخطاء Anthropic للعربية (مفتاح خاطئ، حد مُتجاوَز، خوادم مشغولة، إلخ)
- في `runAI()` تأكدت من تحويل `txt` لـ String قبل `.replace()`

### 📊 تقوية إظهار العقود
**جدول العقود (الأسفل):**
- إضافة عمود **🎯 الهدف** و **عائد متوقع %** 
- إضافة عمود **نقطة التعادل**
- ترتيب جديد: Score أولاً مع شارات 🥇🥈🥉 لأفضل 3
- ألوان متدرجة حسب القوة (>80 أخضر قوي، >60 سماوي، >50 ذهبي)
- زر تتبع أكبر وأوضح

**كرت العقد (في تبويب الإشارات):**
- إطار يتغير لونه حسب Score (أخضر/أزرق/بنفسجي)
- 8 خانات بدل 6: السعر/التعادل/الهدف/العائد/Δ/IV/OI/الحجم
- شارة "Score X/100" مع لون متدرج
- إضافة شارة فركتال في تفصيل المعايير

### 🎯 ظهور الإشارات (كلاسيك/هارمونيك/إيليوت/فركتال)
**قبل:** هارمونيك وإيليوت يظهرون فقط لو وُجد نمط، كلاسيك مخفي.
**الآن:** صف "🎯 الأنماط الفنية" يظهر **دائماً** في تبويب الإشارات بأربعة كروت ملوّنة:
- ✅ كلاسيك (دعم/مقاومة)
- 🦋 هارمونيك (مع اسم النمط واكتماله)
- 🌊 إيليوت (مع رقم الموجة)
- ⚡ فركتال (جديد!)

كل كرت يظهر اللون الخاص به (أخضر/أحمر/رمادي) حسب الإشارة.

## ✨ إضافات جديدة

### ⚡ Williams Fractal — مؤشر فركتال ويليامز
خوارزمية كشف 5-bar pivots للقمم والقيعان:
- يكتشف القمم (bar i.high أعلى من جارَيْن من كل جهة)
- يكتشف القيعان بنفس المنطق
- يحدد الاتجاه: HH+HL = صاعد، LH+LL = هابط
- 4 إشارات ذكية:
  1. السعر قرب آخر قاع → **شراء** (احتمال ارتداد)
  2. السعر قرب آخر قمة → **بيع** (احتمال انعكاس)
  3. كسر صاعد فوق قمة فركتالية → **شراء قوي**
  4. كسر هابط تحت قاع فركتالي → **بيع قوي**

### 🧠 الفركتال يؤثر على ترشيح العقود
في `api/recommend.js` تمت إضافة:
- **مكافأة محاذاة:** إذا فركتال = شراء و عقد CALL → +9 نقاط Score
- **مكافأة إضافية:** إذا الـ Strike قريب من قاع فركتالي → +5 نقاط
- **خصم محاذاة:** إذا الفركتال يخالف اتجاه العقد → -8 نقاط

النتيجة: العقود التي توافق الفركتال تتقدم في القائمة تلقائياً.
